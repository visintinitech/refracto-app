## 📖 Breve descripción

**Refracto** es una herramienta inteligente que analiza tu estilo de escritura y lo descompone en un espectro de métricas clave: **formalidad, complejidad, repetición, riqueza léxica, legibilidad y tono emocional**.

Como un prisma refracta la luz en colores, Refracto revela los componentes invisibles de tu forma de escribir. Ideal para escritores, estudiantes, profesionales y cualquier persona curiosa por entender su propia voz escrita.

**Características principales:**
- ✨ Análisis instantáneo de cualquier texto
- 📊 Visualización clara con gráficos y tarjetas
- 📚 Historial persistente de todos tus análisis
- ⚔️ Comparador lado a lado de dos textos
- 💡 Recomendaciones personalizadas para mejorar
- 🎨 Interfaz oscura elegante y moderna

## 🚀 Demo rápida

```python
# Entrada del usuario
texto = "La verdad es que creo que esto es muy muy interesante."

# Refracto devuelve:
{
  "formalidad": 0.28,      # Informal
  "complejidad": 8.5,      # Oraciones cortas
  "repeticion": 0.67,      # Alta repetición
  "riqueza_lexica": 0.42,  # Vocabulario limitado
  "legibilidad": 72.3,     # Fácil de leer
  "tono": "neutral",       # Sin carga emocional fuerte
  "recomendacion": "🔁 Evita repetir 'muy'. Prueba 'sumamente' o 'extremadamente'."
}
